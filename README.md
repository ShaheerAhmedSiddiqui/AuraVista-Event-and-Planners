# Aura Vista Event & Planner

---

## 📖 Overview

Aura Vista Event & Planner is a desktop-based Event Management System developed using Java Swing and MySQL. The application was built as an Object-Oriented Programming (OOP) semester project to simplify the management of event bookings through an intuitive graphical user interface.

The system enables users to create, view, update, search, and delete event bookings while securely storing customer, venue, and payment information in a relational database.

---

## ✨ Features

- User Login Interface
- Create New Event Bookings
- View All Bookings
- Update Existing Bookings
- Delete Bookings
- Customer Information Management
- Venue Selection
- Event Type Management
- Payment Record Management
- MySQL Database Integration
- Input Validation
- Transaction-based Database Operations
- Interactive Java Swing GUI

## ⚙️ Core Functionalities

### Booking Management
- Add new bookings
- Store customer information
- Select event type and venue
- Record payment details
- Validate user inputs

### Booking Records
- Display all bookings
- View customer information
- View payment details
- Display event information

### Update Records
Users can update:
- Event Date
- Event Type
- Venue
- Booking Price
- Payment Method

### Delete Bookings
Bookings can be removed using their Booking ID.

---

## 🗄 Database

The application uses MySQL as the backend database.

Major tables include:
- Customers
- EventBooking
- EventTypes
- Venues
- Payments

The system maintains relationships between these tables using foreign keys to ensure data consistency.
